# EnigmaBudget
## Aplicación para el seguimiento de gastos y finanzas personales

Proyecto personal para el seguimiento de gastos y finanzas, basado en principio en una interfaz desarrollada en Angular, y un servicio API REST en .NET Core.

## Funcionalidades deseadas:

    - Seguimiento de Gastos 💸
    - Seguimiento de Ahorros y Plazos Fijos 💹
    - Seguimiento de deudas 💰
    - Control de Gastos con Tarjetas de Crédito 💳
    - Previsión de gastos futuros 🗓️
    - Soporte multiplataforma 🤳🏻
    - Soporte de múltiples idiomas 📚
